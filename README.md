# Primenum
A prime number is a natural number greater than 1 that has only two factors: 1 and itself. Examples of prime numbers include 2, 3, 5, and 7. To check whether a number is prime, we can use a simple program that tests if the number is divisible by any value from 2 up to its square root. If it is divisible by any of these numbers, it is not a prime number; otherwise, it is prime.
